This folder needs the following images for the game to display properly:

1. happy.jpg - An image showing a happy father-in-law
2. neutral.jpg - An image showing a neutral/unsure father-in-law 
3. angry.jpg - An image showing an angry father-in-law

You can use photos, illustrations, or any images that represent these emotions appropriately.

The recommended size is around 500x300 pixels, but any reasonable size will work.
